## [router](https://v3.router.vuejs.org/installation.html#direct-download-cdn)

```
npm install vue-router
```
OR
```
vue add router
```
- 選擇 history mode

### Started
- main.js 新增
```
import router from './router'

const app = new Vue({
  router
}).$mount('#app')
```
